# Capítulo 01: Introducción al Desarrollo Web #

## 01.01 - Entendiendo lo Básico ##

### 01.01.01 - ¿Qué es el Desarrollo Web? ###

Es un campo dinámico y en constante evolución que comprende un rango de actividades involucradas en la construcción y mantenimiento de sitios web y aplicaciones web.

En este contexto, nos referiremos a desarrollador web al proceso de crear, diseñar y mantener sitios web y aplicaciones web. Involucra una aproximación multidisciplinaria que incluye desarrollo _frontend_, _backend_ y la integración de varias tecnologías para hacer la entrega y que la experiencia web sea funcional.

#### Evolución del Desarrollo Web ####

Inicialmente, la páginas HTML estáticas dominaban la web, ofreciendo interactividad limitada.

Gracias a JavaScript como un poderoso lenguaje _script_ revolucionó el _frontend_, posibilitando la interacción y haciendo más dinámica la web. Con el crecimiento de lenguajes para el lado-servidor como **PHP**, **Python** y **Ruby**, las aplicaciones web se hicieron más sofisticadas, permitiendo el procesamiento de información y crear lógica del lado-servidor.

La evolución continuó con la introducción de _frameworks_, como **Angular**, **React** y **Vue.js** que mejoran el desarrollo del _frontend_. En el _backend_, _frameworks_ como **Django**, **Flask**, **Ruby and Rails** y **Node.js** proveen a los desarrolladores las herramientas para construir aplicaciones escalables e eficientes del lado-servidor.

#### La Importancia en la Era Digital ####

Los sitios web y aplicaciones web sirven como interfaz primaria entre negocios, organizaciones y usuarios.

### 01.01.02 - El Rol del HTML y CSS ###

#### Vistazo al HTML (_Hypertext Markup Language_) ####

El HTML, la columna vertebral del desarrollo web, es un lenguaje de etiquetado que estructura el contenido en la web. Utiliza un sistema basado en etiquetas, donde las etiquetas definen los elementos, como encabezados, párrafos, imágenes y enlaces. El HTML es el núcleo en la creación de la estructura de una página web. Aquí listamos los aspectos clave:

- Elementos y Etiquetas HTML
- Las etiquetas HTML sirven para construir bloques, encapsular contenido y proveer estructura. Elementos como `<head>`, `<body>`, `<div>` y etiquetas semánticas como `<article>`, `<section>`, enriquecen el significado del documento. Entender el rol de cada etiqueta es fundamental para la creación de un contenido bien estructurado.
- Atributos y Estructura para el Documento
- Atributos que mejoran los elementos HTML, proveyendo información adicional o funcionalidad. Exploraremos dichos atributos como `class`, `id` y `src`, y como ellos contribuyen a dar estilo e interactividad. Adicionalmente, discutiremos la estructura general del documento, incluyendo los elementos `<!DOCTYPE>`, `<html>`, `<head>` y `<body>`.

#### Vistazo al CSS (_Cascading Style Sheets_) ####

El CSS, es la contraparte estilista del HTML, habilita la presentación y disposición del contenido web. Aplica selectores para apuntar a elementos HTML y aplicar estilos, creando una apariencia visual y diseño consistente.

Selectores y Declaraciones. Los selectores definen los elementos a los que apunta, y las declaraciones especifican los estilos aplicados a esos elementos.

_Box Model_ y _Latyout_. Entender el modelo de caja es crucial para diseñar disposiciones. Profundizaremos en conceptos como margenes, espaciado, bordes y como contribuyen en el tamaño y espaciados de los elementos. Las propiedades de disposición como `display`, `position` y `float` serán discutidas para lograr un diseño responsivo.

#### Significa de Estructura y Estilo en el Contenido Web ####

El HTML establece la estructura del documento, definiendo la jerarquía de su contenido, mientras que CSS añade atractivo visual, asegurando una presentación agradable y consistente.

### 01.01.03 - Vistazo del Proceso de Desarrollo Web ###

El desarrollo web es dinámico y un campo multifacetico, compuesto por varias etapas e especializaciones. Entender el proceso general es crucial para desarrolladores principiante y veteranos.

#### Desarrollo _Frontend_ vs _Backend_ ####

El desarrollo de _frontend_ involucra la creación de interfaces de usuario y la experiencia de usuario que interactuaran. Emplean tecnologías como HTML, CSS y JavaScript.

El desarrollo del _backend_ se enfoca en el lado-servidor de aplicaciones, crean las base de datos, administran la lógica del servidor y el flujo de información. Lenguajes de programación como **Python**, **Ruby** y **Node.js** son los más utilizados.

#### Desarrollo _Full Stack_ ####

Ellos poseen experiencia tanto en _frontend_ como _backend_, haciéndolos contribuidores versátiles en el proceso entero de desarrollo.

#### Colaboración y Flujo de Trabajo ####

El desarrollo de proyectos web comúnmente involucran colaboración con equipos diversos o individuos con diferentes habilidades.

En esta sección veremos tecnologías y metodologías como un sistema de control de versiones, herramientas de administración de proyectos y plataformas de comunicación.
